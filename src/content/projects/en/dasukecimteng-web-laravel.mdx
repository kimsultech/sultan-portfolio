## Overview

In March-May 2024, I did a college assignment, namely PKL or Internship at a Government Agency. Then there I was assigned to create a website about MSMEs (UMKM).\
The DASUKECIMTENG website is a website that focuses on the MSME program in Cimahi Tengah District. Developed using Laravel 10 and the Mysql Database and Flowbite Tailwind to make it easier to create web designs.
