## Overview

In this project, I had the opportunity to develop a Native Mobile Application (Android) Using Java Language and for the Web and API sections I used Express JS and MYSQL Database and Equipment that facilitates Development such as Bootstrap, AdminLTE, etc.\
ICore Disdukcapil is an Application created to facilitate the Community in Submitting or Managing Documents related to Disdukcapil, In addition to the Covid conditions in previous years which limited the Community to visit the Disdukcapil office, Pangkajene and the Islands (Pangkep) is also a district whose regional structure consists of 2 main parts that form this district, namely the land area and the sea area.
