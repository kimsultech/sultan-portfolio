## Ikhtisar

Dalam proyek ini, saya berkesempatan untuk mengembangkan Aplikasi Mobile (Android) Native dengan Menggunakan Bahasa Java dan untuk bagian Web dan API saya menggunakan Express JS serta Database MYSQL dan Perlengkapan yang memudahkan dalam Pengembangan seperti Bootstrap, AdminLTE, dll.\
ICore Disdukcapil merupakan Aplikasi yang dibuat untuk memudahkan Masyarakat dalam Pengajuan atau Pengurusan Dokumen terkait Disdukcapil, Selain karena kondisi Covid tahun-tahun sebelumnya yang membuat terbatasnya Masyarakat untuk mendatangi kantor Disdukcapil, Pangkajene dan Kepulauan (Pangkep) juga merupakan kabupaten yang struktur wilayah terdiri atas 2 bagian utama yang membentuk kabupaten ini yaitu wilayah daratan dan wilayah lautan.
