## Ikhtisar

Pada Bulan Maret-Mei 2024 saya melakukan tugas kuliah yaitu PKL atau Magang di suatu Instansi Pemerintahan, Lalu disana saya ditugaskan untuk membuat Website tentang UMKM.\
Web DASUKECIMTENG merupakan web yang berfokus pada program UMKM di Kecamatan Cimahi Tengah, Dikembangkan menggunakan Laravel 10 dan Database Mysql serta Flowbite Tailwind untuk mempermudahkan dalam membuat desain Web.
